Estágio de Desenvolvimento - Oak Tecnologia

Descrição do Projeto

Desafio de criação de um CRUD como parte do processo seletivo para a vaga de estágio Back-End na empresa OAK. O projeto foi desenvolvido utilizando JavaScript, CSS e HTML, com armazenamento de dados no Local Storage, garantindo a persistência das informações na aplicação. A solução resultou em uma aplicação web totalmente funcional, permitindo a criação, leitura, atualização e exclusão de dados. O foco esteve na implementação de uma interface amigável e responsiva, além de boas práticas de codificação, organização e estruturação do código.
